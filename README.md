Coins
-----------------
### 游戏介绍:
翻金币游戏，基于Qt平台开发的小游戏

游戏介绍：
翻金币项目是一款经典的益智类游戏，我们需要将金币全部都翻成金色，才视为胜利。

游戏分为4个场景：
开始界面 、选择难度界面、 选择关卡界面 、 游戏场景界面
